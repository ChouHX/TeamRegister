从 HAR 里还原的**获取 `__Secure-next-auth.session-token` 的完整关键请求链**如下（只保留核心接口 + 关键参数）。

---

# 一、完整请求链

```
1 发送邮箱验证码
POST https://auth.openai.com/api/accounts/passwordless/send-otp

2 验证邮箱验证码
POST https://auth.openai.com/api/accounts/email-otp/validate

3 OAuth 回调
GET https://chatgpt.com/api/auth/callback/openai

4 页面跳转
GET https://chatgpt.com/

返回 Set-Cookie:
__Secure-next-auth.session-token
```

---

# 二、详细接口

## 1 发送验证码

### Request

```
POST https://auth.openai.com/api/accounts/passwordless/send-otp
```

### 作用

向邮箱发送 OTP

### 返回关键 Cookie

```
auth_provider
login_session
hydra_redirect
oai-client-auth-session
```

其中最关键的是：

```
login_session
```

后续登录流程依赖它。

---

# 2 验证 OTP

### Request

```
POST https://auth.openai.com/api/accounts/email-otp/validate
```

### Body

```json
{
  "code": "971880"
}
```

### Header 必须带

```
cookie:
login_session=...
auth_provider=...
```

### 返回 Cookie

```
auth_provider
login_session
hydra_redirect
oai-client-auth-session
auth-session-minimized
```

此时**用户已经完成身份认证**

---

# 3 OAuth Callback

### Request

```
GET https://chatgpt.com/api/auth/callback/openai
```

### Query

```
code=ac_0-qiJyd54DJswbjSSitLUxhHGy7O90rkKwQDJMZEKDU.bv286qKiwP9iPDPUyAXfyw_pN1i7BPkGUKCTOHHRB1c

scope=openid email profile offline_access model.request model.read organization.read organization.write

state=vPxofi7JtikzhKeszkC_ETscZRHUfIUddWk20LR-Ifw
```

### Response

```
302 Redirect
```

### 关键返回 Cookie

```
Set-Cookie:
__Secure-next-auth.session-token=xxxxx
```

同时返回

```
_account
oai-hlib
```
